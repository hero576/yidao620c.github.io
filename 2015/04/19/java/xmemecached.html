使用xmemcached客户端

发表于 2015-04-19 | 分类于 java | | 阅读次数

Xmemcached是基于java nio实现的高性能可扩展的memcached客户端。它的主要特点：

  1. 高性能，稳定可靠，已经在众多公司的众多项目里得到应用。
  2. 功能完备：客户端分布式、权重、最新最完整的协议支持。
  3. 可扩展，易于集成
  4. 可动态增删memached节点
  5. 客户端操作统计
  6. NIO连接池

等等，更多信息请见wiki文档。
